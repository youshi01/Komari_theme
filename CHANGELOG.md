# 更新日志

## v1.1.8-p1

- 优化仪表盘“断点扫描”动效：扫描拖尾不再跟随实时百分比重置，减少全环仪表刷新后掉帧和同步跳动。
- 为每张节点卡、每个仪表指标加入稳定扫描相位，让扫描动效更连续、不容易全站同拍。
- 首页“卡片与样式”快捷面板新增“应用全局”按钮，仅登录管理员可见，可将当前卡片与样式保存为全站默认。

## v1.1.8

- 优化跑马灯与历史条 Canvas 绘制：缓存颜色解析与 CSS 变量解析，降低多节点同时动画时的主线程压力。
- Canvas 动画现在会在页面隐藏或节点离开可视区域附近时暂停重绘，并限制高 DPR 下的像素绘制量，让动态衔接更稳。
- 主题管理页的跑马灯预览改为静态绘制，后台 Ping 任务和节点列表改为空闲/用户操作时再加载，减少打开全局设置时的卡顿。
- 保存主题设置时仅在背景配置变更后处理上传背景，并在大体积配置 JSON 估算超过阈值后提前短路，减少保存前后的卡顿感。

## v1.1.7

- 调整“霓虹警戒”跑马灯配色，让对比和亮度更明显。
- 新增 8 套跑马灯配色预设：赛博霓虹、电光蓝紫、落日能量、酸性绿光、等离子、熔岩警戒、矩阵绿、极光撞色。
- 新配色会同时用于首页快捷面板和主题管理页，仍然支持在预设基础上继续自定义每项指标颜色。

## v1.1.6

- 重做液位容器的科技 HUD 细节：水波圆球、竖向液柱、分段胶囊、横向胶囊舱、椭圆透镜和六边晶核加入轨道、扫描、刻度、能量线和核心脉冲。
- 新增液位面板扫光、轨道流动、数据包移动、晶核呼吸和分段闪烁等动态效果，并继续跟随“动效强度”设置。
- 优化液位容器的玻璃、发光、网格和边框质感，整体更接近概念图里的未来监控面板风格。

## v1.1.5

- 修复“同步卡片色彩”对部分卡片外壳不生效的问题；开启后节点卡片会稳定跟随当前渐变背板色彩。
- 玻璃、霓虹、柔彩、极简和复古 CRT 外壳现在都会在同步色彩开启时改用渐变背板的主色、辅色和强调色。
- 同步优化节点卡片内部指标块、条形卡片信息块、仪表/液位容器、标签和详情按钮的背景与边框，让总览和节点卡片的视觉更统一。

## v1.1.4

- 重新调整手机端条形卡片：窄屏下不再把条形内容压缩进屏幕宽度，而是使用固定宽度横向卡片，外层支持横向滑动，保留条形视觉。
- 手机端条形卡片改为左侧节点信息、右侧两排指标与状态，避免条形卡片被压成方形。

## v1.1.3

- 首页快捷按钮改为每次打开页面默认收起，避免遮挡首页数据和手机端内容。
- 优化手机端条形卡片布局：条形卡片在窄屏下不再两列排列，核心指标和状态信息改成单列横向小条，避免显示成方形卡片。

## v1.1.2

- 修复“未登录隐藏节点”在已登录状态下仍被主题隐藏的问题。`hidden` 字段现在按接口语义仅用于未登录访客过滤，管理员/已登录用户可以正常看到这些节点。
- 同步修正首页节点列表、详情页节点读取、Ping 首页概览和实时排序里的隐藏节点判断，避免不同视图表现不一致。

## v1.1.1

- 在“卡片外壳”设置中新增“卡片形态”，支持“方型卡片”和“条形卡片”两种首页节点展示方式。
- 条形卡片采用单列紧凑列表，横向展示节点身份、CPU/内存/硬盘/负载、上下行速率、延迟、丢包、到期和在线时长，适合服务器数量较多时快速扫读。
- 首页快捷面板与主题管理页均支持保存卡片形态，本机外观和管理员全站默认都可独立生效。
- 条形卡片同步适配数据面板、清透玻璃、霓虹暗面、柔和彩块、极简白板和复古 CRT 外壳，并补充响应式布局。

## v1.1.0-p9

- 优化主题管理页「首页服务器排序」：自定义服务器顺序列表默认收起，只显示摘要、恢复默认排序和展开编辑按钮，避免服务器较多时页面过长。
- 展开自定义排序后仍保留搜索、拖拽排序、置顶、上移、下移和置底操作。
- 优化「主页延迟检测」绑定：每个 Ping 任务新增「应用全部节点」按钮，可一键将所有服务器绑定到该任务，并自动从其他任务移除。
- 已绑定全部服务器的 Ping 任务会禁用一键应用按钮，减少误操作。

## v1.1.0-p8

- 重做液位容器的个性层，让 8 种形态不再只是换轮廓，而是带有不同的未来 HUD 视觉。
- 水波圆球新增轨道环、扫描扇面和节点脉冲；横向胶囊舱新增双轨传输、能量包和信号折线。
- 竖向液柱新增刻度、能量脊线和上升脉冲；椭圆透镜新增全息网格、准星与扫描光。
- 分段胶囊新增状态格、流动中线和传输点；六边晶核新增晶体切面、核心多边形和节点光。
- 悬浮水滴新增双轨环绕与内部星形能量；环形液舱新增内外轨道、方向信标和节点脉冲。
- 优化液位预览和动画兼容性，纹理强度会影响科技层明显程度，关闭系统动画时同步停用新动效。

## v1.1.0-p7

- 为信息展板「液位容器」新增 8 种形态：水波圆球、横向胶囊舱、竖向液柱、椭圆透镜、分段胶囊、六边晶核、悬浮水滴、环形液舱。
- 首页快捷面板和主题管理页均支持调整液位容器形态，并可配置波浪高度、玻璃强度、光晕强度、动效强度和纹理强度。
- 光晕强度与动效强度支持 200% 上限，和环形仪表的增强区间保持一致。
- 优化液位容器的 SVG 外壳、高光、分段线、环形核心和预览反馈，让不同形态的视觉差异更明显。

## v1.1.0-p6

- 新增信息展板「液位容器」，CPU、内存、硬盘、负载、上下行、延迟和丢包统一使用水波液位球展示。
- 液位容器中心显示当前映射百分比，底部保留真实数值；上下行会按当前速率单位自动映射到合适上限，延迟继续使用可配置上限。
- 新增卡片外壳「复古 CRT」，包含暗色荧光屏、扫描线、暗角和轻微色散质感，内部指标块与液位球同步适配。
- 更新首页快捷面板与主题管理页预览，让液位容器和复古 CRT 在选择时有对应视觉反馈。

## v1.1.0-p5

- 优化环形样式选择面板为紧凑两列，样式变多后首页快捷面板和主题管理页仍然可滚动选择。
- 修正首页快捷面板中环形样式列表的紧凑布局位置，避免误作用到数据条样式列表。

## v1.1.0-p3

- 新增清透光环、霓虹双轨、分段刻度、柔光厚环、极简细线 5 组环形样式预设，弧光、全环和指针仪表共用同一套样式体系。
